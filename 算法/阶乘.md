### 阶乘

#### 一、问题：
求一个数a的阶乘，那么，a!=a\*(a-1)\*(a-2)\*(a-3)\*……\*2\*1.比如，6的阶乘6！=6\*5\*4\*3\*2\*1=720.那么，如何通过php代码实现求任意一个数的阶乘？
#### 二、实现代码：

```php
<?php
function demo($a) {
  if ($a > 1) {
    $r = $a * demo($a - 1);
  } else {
    $r = $a;
  }
  return $r;
}
$a = 6;
echo $a . "的阶乘的值" . demo($a);
?>
```
